### ЗАПУСК САЙТА В ПРОДАКШЕН

```
Убедитесь, что файл `.env` настроен с необходимыми переменными окружения (можно использовать `.env.example` как шаблон).
Перед запуском проекта, также убедитесь что этап СБОРКА И ПРЕВЬЮ работают корректно и не выдают ошибок
```

```
Стяните последние изменения с Github :
git pull
```
```
Запустите проект с помощью docker-compose :
docker-compose -f docker-compose.prod.yml up -d
```

```
После запуска сервисы будут доступны по следующим адресам:
   - Frontend: http://localhost:80
   - Directus API: http://localhost:8055
```


```
Если вы собираетесь вносить правки или хотите посмотреть фронтенд то запустите этапы УСТАНОВКА + РАЗРАБОТКА, или УСТАНОВКА + СБОРКА + ПРЕВЬЮ
```
---------------------------------------------------------------------
---------------------------------------------------------------------

### ЭТАП УСТАНОВКА

```bash
# npm
npm install
```
### ЭТАП РАЗРАБОТКА `http://localhost:3000`:

```bash
# npm
npm run dev
```
### ЭТАП СБОРКА

```bash
# npm
npm run build
```
### ЭТАП ПРЕВЬЮ

```bash
# npm
npm run preview
```
